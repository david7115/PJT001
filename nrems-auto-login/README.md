# NREMS 자동로그인 앱 (Android)

여러 발전소의 NREMS 모바일 사이트(`http://www.nrems.co.kr/m/login.php`)에
매번 아이디/비밀번호를 입력하는 불편을 없애기 위한 안드로이드 앱입니다.

## 주요 기능

- **발전소 계정 여러 개 등록**: 발전소 이름 + 아이디 + 비밀번호를 등록해 목록으로 관리
- **탭 한 번으로 자동 로그인**: 계정을 누르면 로그인 페이지가 열리고 아이디/비밀번호가 자동
  입력·제출됩니다
- **로그인 후 자동 이동(선택)**: 계정마다 "로그인 후 이동할 페이지 URL"을 지정하면, 로그인
  성공 직후 해당 모니터링 페이지로 바로 이동합니다
- **계정 전환 지원**: 로그인 실행 시 이전 세션 쿠키를 비워서 발전소를 바꿔가며 확인할 수 있습니다
- **암호화 저장**: 아이디/비밀번호는 Android Keystore 기반
  EncryptedSharedPreferences(AES-256)로 기기 안에만 암호화 저장됩니다. 외부 서버로 전송되지
  않습니다.

## 폰에서 테스트하는 방법 (Android Studio 없이)

이 저장소에는 GitHub Actions 빌드 워크플로
(`.github/workflows/nrems-auto-login-apk.yml`)가 포함되어 있습니다.

1. GitHub 저장소 → **Actions** 탭 → "Build NREMS Auto Login APK" 워크플로 선택
2. 최근 실행(또는 **Run workflow** 버튼으로 수동 실행) → 완료 후 하단
   **Artifacts**에서 `nrems-auto-login-debug-apk` 다운로드
3. 압축을 풀면 나오는 `app-debug.apk`를 폰으로 옮겨 설치
   (설치 시 "출처를 알 수 없는 앱 허용"을 켜야 합니다)
4. 앱 실행 → "발전소 계정 추가" → 아이디/비밀번호 입력 → 목록에서 로그인 버튼 탭

Android Studio가 있다면 `nrems-auto-login` 폴더를 열어 그대로 빌드/실행하면 됩니다
(minSdk 24, Android 7.0 이상).

## 자동 로그인이 동작하는 원리

`WebViewActivity.kt`가 로그인 페이지 로딩 완료 시 자바스크립트를 주입합니다.

1. 한국 PHP 사이트에서 흔한 필드명(`id`/`pw`, `mb_id`/`mb_password`,
   `user_id`/`passwd` 등)을 먼저 찾고
2. 없으면 `input[type=password]`를 기준으로 아이디 칸을 휴리스틱하게 찾은 뒤
3. 값을 채우고 로그인 버튼(또는 폼 submit)을 자동 실행합니다

무한 재시도를 막기 위해 자동 제출은 최대 2회까지만 시도하며, 실패하면
"자동 로그인에 실패했습니다" 안내가 표시되고 수동 입력이 가능합니다.

### 필드 감지가 실패할 경우

사이트 로그인 폼이 특이한 구조라면 `WebViewActivity.kt`의 `AUTO_LOGIN_JS` 안
`idSelectors` / `pwSelectors` 배열에 실제 필드명을 추가하면 됩니다.
(폰 크롬에서 로그인 페이지 열기 → 메뉴 → 데스크톱 모드 → 페이지 소스 보기로
`<input name="...">` 값을 확인할 수 있습니다.)

## 참고 사항

- NREMS 사이트가 HTTP(비암호화)라서 매니페스트에 `usesCleartextTraffic=true`가
  설정되어 있습니다. 통신 자체는 사이트 특성상 평문이므로 공용 와이파이에서는 주의하세요.
- iOS(아이폰)는 별도 개발이 필요합니다. 현재 버전은 안드로이드 전용입니다.
- 다음 단계로 확장 가능: 로그인 후 발전량 등 필요한 모니터링 수치만 추출(스크래핑)해서
  한 화면에 발전소 전체 현황을 요약해 보여주는 기능. 필요하면 요청해 주세요.
